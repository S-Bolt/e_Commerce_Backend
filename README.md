# e_Commerce_Backend
This is a backend e-commerce app that uses RESTful API for managing products, categories, and tags.  Users can create, read, update, and delete entities in the databse using  the APIs.

## Technologies Used
- Node.js
- Express.js
- Sequelize
- PostgresSQL (Database)
